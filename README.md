# Game of Life

Jeg kodet spillet Game of Life i python i mitt første semester. Når du kjører hovedprogrammet blir du spurt i terminal hvor mange rad og kolonner du vil få når du spiller. 
Les mer om hvordan Game of Life fungerer her: https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life

Slik fungerer det:
Levende celler er representert som 'O' og døde celler er representert som '.'.
Generasjonsnummeret og antall levende celler vises under brettet.
For å fortsette, trykker man enter. For å avslutte trykker man 'q' og så enter.
